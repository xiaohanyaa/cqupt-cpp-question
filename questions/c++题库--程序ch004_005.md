## ch004_005
``` c++
1) #include <iostream>
2) #include <iomanip>
3) using namespace std;
4) void main()
5) {
6)   int s = 0, i = 1, n;
7)   cin >> n;
8)   while (i++ <= n)
9)     s += i;
10)   cout << s;
11) }

```
### 题目22846
问题：1.执行该程序，从键盘输入3，循环执行了几次？
A.4
B.3
C.2
D.1


**解析如下：**

------

解析：
该程序的功能是计算从1到n的累加和。输入n=3时，循环的条件是`i++ <= n`，即`i`从1开始，每次循环后`i`加1，直到`i`大于n时结束循环。

- 初始`i=1`，进入循环，`i`变为2，`s`加上2。
- `i=2`，进入循环，`i`变为3，`s`加上3。
- `i=3`，进入循环，`i`变为4，`s`加上4。
- `i=4`，不满足`i<=n`，循环结束。

因此，循环执行了3次。

{'22846':'B'}
