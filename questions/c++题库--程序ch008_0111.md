### 题目24177
问题：10.　　下列代码与7~12行程序段等价的是（　　　　）：
A.　　while　(i　<　count　-　1)｛a[++i]　=　cin.get();｝
B.　　while　(i　<=　count)｛a[i++]　=　cin.get();｝
C.　　for(;i　<　count　-　1;i++)｛a[i]　=　cin.get();｝
D.　　for(;i　<=　count　-　1;)｛a[i--]　=　cin.get();｝


**解析如下：**

------

解析：
原程序段（第 7~12 行）的功能是：
1. 使用 `while` 循环从键盘读取字符，存储到数组 `a` 中。
2. 循环条件是 `i < count - 1`，即 `i < 9`。
3. 每次循环中，`a[i]` 被赋值为 `cin.get()` 的返回值，然后 `i` 自增 1。
4. 循环结束后，`a[i]` 被设置为 `0`。

选项分析：
- A: `while (i < count - 1) { a[++i] = cin.get(); }`，这里 `++i` 会先自增再使用，导致第一个字符被存储在 `a[1]`，不符合原程序。
- B: `while (i <= count) { a[i++] = cin.get(); }`，循环条件错误，会导致数组越界。
- C: `for(; i < count - 1; i++) { a[i] = cin.get(); }`，这与原程序的功能完全一致。
- D: `for(; i <= count - 1; ) { a[i--] = cin.get(); }`，这里 `i--` 会先使用再自减，且循环条件错误，不符合原程序。

等价的代码是选项 C。

正确答案：C

{'24177':'C'}
