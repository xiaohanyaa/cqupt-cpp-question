问题：程序执行完第24行后，maximum()函数的返回值是：

A、77
B、90
C、96
D、81


**解析如下：**

------

解析：`maximum` 函数的目的是找出二维数组中的最大值。通过遍历整个数组，`max` 变量不断更新为当前遇到的最大值。

根据数组 `sg` 的初始化值：
```
{ {68, 77, 73, 86},
  {87, 96, 78, 89},
  {90, 70, 81, 86} }
```
我们可以看到最大值是 96。

遍历过程如下：
- 初始 `max` 为 0。
- 第一行最大值更新为 86。
- 第二行最大值更新为 96。
- 第三行没有比 96 更大的值。

因此，执行完第24行后，`maximum` 函数的返回值是 96。
