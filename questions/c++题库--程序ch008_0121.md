### 题目24186
问题：9.  下列程序语句，与第8行代码等价的是（  ）：
A.  cout << language[i++][j];
B.  cout << *(language+i)[j++];
C.  cout << (*language+i)[j++];
D.  cout << *(language[i]) + j;


**解析如下：**

------

解析：
在代码的第8行，表达式 `cout << language[i][j++]` 输出 `language[i]` 指向的字符串中的第 `j` 个字符，并将 `j` 递增。等价的表达式应该是 `cout << (*(language + i))[j++]`，即先通过 `*(language + i)` 获取 `language[i]` 指向的字符串，然后输出该字符串中的第 `j` 个字符，并将 `j` 递增。

{'24186':'B'}
