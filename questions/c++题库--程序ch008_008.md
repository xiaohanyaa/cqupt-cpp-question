A.  c
B.  cout函数
C.  cout对象
D.  无返回值


**解析如下：**

------

解析：

2. `std::cout` 是一个输出流对象，用于将数据输出到标准输出设备（通常是控制台）。
3. `<<` 运算符在 `std::cout` 中被重载，用于将数据插入到输出流中。
4. `std::cout << c << endl;` 的返回值是 `std::cout` 对象本身，以便支持链式调用。

因此，程序第27行语句 `cout << c << endl` 的返回值是 `std::cout` 对象。
